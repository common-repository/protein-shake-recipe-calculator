== Description ==

Allows the user to calculate her optimum **post-workout protein shake** recipe and determine right mixture of these three essential ingredients, plus her daily recommended protein dose:

1. Protein
1. Carbohydrate
1. Water

All calculations are based on her body weight.

= Credits =

The recipe's physiological foundation courtesy of [Whey Protein Institut](http://whey-proteine.org/). German translation by [PHP-Entwickler Robert Wetzlmayr](http://wetzlmayr.ch/).

== Installation ==

1. Unzip the downloaded ZIP file and upload the contained files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

= Add the sidebar widget =

1. Install the **Protein Shake Recipe Calculator** widget through the 'Design' menu in WordPress

== Frequently Asked Questions ==

= Which prerequisites does the Protein Shake Recipe Calculator widget expect? =

The Protein Shake Recipe Calculator widget requires JavaScript for its calculations.
